How to create a token
  1. Go to GitHub Token Settings
  2. Click "Generate new token (classic)"
  3. Give it a descriptive name (e.g., "GitHub Actions Folder Organizer")
  4. Select scope: repo (for full access)
  5. For fine-grained tokens, select: actions:read and contents:read
  6. Generate and copy the token
  7. Paste it above and click "Save Token"

Security: Your token is stored securely in Chrome's encrypted storage and never shared. You can revoke it anytime from GitHub settings.

About

This extension organizes your GitHub Actions workflows into custom folders based on a configuration file in your repository (.github/actions-folders.json).

Configuration File Format

{
  "folders": [
    {
      "name": "Build & Test",
      "workflows": [
        "ci.yml",
        "test.yml"
      ]
    },
    {
      "name": "Deployment",
      "workflows": [
        "deploy-prod.yml",
        "deploy-staging.yml"
      ]
    }
  ]
}

Features

  • Organize workflows into collapsible folders
  • Works with public and private repositories
  • Automatic caching for better performance
  • Fallback to default GitHub UI if config is missing
  • Matches GitHub's design system
