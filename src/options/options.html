About

This extension organizes your GitHub Actions workflows into custom folders based on a configuration file in your repository (.github/actions-folders.json).

Configuration File Format

{
  "folders": [
    {
      "name": "Build & Test",
      "workflows": [
        "ci.yml",
        "test.yml"
      ]
    },
    {
      "name": "Deployment",
      "workflows": [
        "deploy-prod.yml",
        "deploy-staging.yml"
      ]
    }
  ]
}

Features

  • Organize workflows into collapsible folders
  • Works with public and private repositories
  • Automatic caching for better performance
  • Fallback to default GitHub UI if config is missing
  • Matches GitHub's design system
